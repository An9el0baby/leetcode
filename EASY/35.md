# [35. Search Insert Position](https://leetcode.com/problems/search-insert-position)

## 解法1 [left,right)

- python

```python
class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        left = 0 
        right = len(nums)
        while left < right:
            middle = (left + right) // 2
            if nums[middle] > target:
                right =  middle
            elif nums[middle] < target:
                left = middle + 1 
            else:
                return middle
        return left
```
